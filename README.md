# 医护人员排班系统 python869

## 项目概述

本项目是基于 SpringBoot 开发的医护人员排班系统，旨在帮助医疗机构高效、合理地管理医护人员的排班工作。

## 技术栈

- **架构**: B/S、MVC、Web框架
- **开发环境**：IDEA、JDK1.8、MySQL、Maven、Navicat
- **技术栈**：
  - 后端：Java、SpringBoot、MyBatis
  - 前端：HTML、CSS、Vue

## 功能模块

### 系统角色

- 管理员
- 医护人员

### 管理员功能

- 个人中心
- 医院信息管理
- 医护信息管理
- 医护类型管理
- 排班信息管理
- 排班类型管理
- 科室信息管理
- 投诉信息管理

### 医护人员功能

- 修改个人信息
- 查看个人排班信息
- 查看收藏信息

## 运行环境

- 服务器：推荐使用支持 Java 和 MySQL 的服务器环境
- 客户端：现代浏览器

## 部署步骤

1. 环境搭建：确保服务器环境满足开发环境要求
2. 数据库配置：导入 SQL 文件到 MySQL 数据库
3. 代码部署：将后端代码导入 IDEA 或其他支持 Maven 的 IDE 中，并构建部署
4. 前端部署：将前端代码放置于 Web 服务器指定目录

## 目录结构

```
/backend                # 后端代码目录
    /src
        /main
            /java
            /resources
    /pom.xml
/frontend               # 前端代码目录
    /src
        /assets
        /components
        /views
    /index.html
```

## 核心亮点

- 使用 SpringBoot 简化配置，提高开发效率
- 前后端分离，易于维护和扩展
- 功能模块清晰，便于不同角色的使用
